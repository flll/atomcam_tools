Get started in minutes

導入は 5 ステップ

母艦 PC もアプリのインストールも不要。SD カードにコピーして挿すだけです。

SD カードは FAT-FS でフォーマットしてください(exFAT は u-boot 非対応)。Class 10 以上を推奨。自己責任でご利用ください — メーカーへの問い合わせはお控えください。

Security

開放されるポート

WebUI 80 SSH 22 mDNS 5353 Samba 137-139,445 RTSP 8554,8080 WebRTC 8555

映像・JPEG・WebUI は LAN 内からアクセス可能です。LAN のセキュリティは各自で確保してください。SSH は SD カードに公開鍵を書き込む方式です。
